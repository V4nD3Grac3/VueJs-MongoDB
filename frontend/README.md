# frontend

## Create .env in root diretory frontend
```
VUE_APP_BACKEND_URL=http://localhost:3000                      # Replace this with your backend url
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
